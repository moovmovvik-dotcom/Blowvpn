# BlowVPN Web Version

Веб-версия BlowVPN - современный VPN клиент для iOS, Android и десктопа.

## 🚀 Быстрое развёртывание

### Вариант 1: Vercel (рекомендуется)

```bash
# Установите Vercel CLI
npm i -g vercel

# Разверните проект
vercel
```

**Ссылка на развёрнутый проект будет выглядеть:** `https://blowvpn.vercel.app`

---

### Вариант 2: GitHub Pages (бесплатно)

1. Перейдите в **Settings** → **Pages**
2. Выберите **Deploy from a branch**
3. Branch: `main`, Folder: `/ (root)`
4. Сохраните

**Ссылка:** `https://moovmovvik-dotcom.github.io/Blowvpn/`

---

### Вариант 3: Netlify

```bash
# Установите Netlify CLI
npm i -g netlify-cli

# Разверните
netlify deploy --prod
```

## 📱 Использование на iOS

### Способ 1: Add to Home Screen (рекомендуется)
1. Откройте в Safari: `https://ваш-домен.com`
2. Нажмите **Share** (значок со стрелкой)
3. Выберите **Add to Home Screen**
4. Назовите приложение: **BlowVPN**
5. Нажмите **Add**

✅ Приложение появится на домашнем экране как полноценное приложение!

### Способ 2: Ярлык прямого доступа
1. Добавьте ярлык на Siri Shortcuts
2. Откройте сайт в полноэкранном режиме

## 🛠️ Технические детали

### PWA (Progressive Web App)
- **Manifest.json** - данные для установки
- **Service Worker** - кеширование и работа оффлайн
- **Responsive Design** - адаптивный дизайн для всех экранов

### Поддерживаемые протоколы:
- ✅ VLESS (`vless://...`)
- ✅ VMESS (`vmess://...`)
- ✅ Shadowsocks (`ss://...`)

### Хранилище данных:
- Подписки сохраняются в `localStorage`
- Нет облачного сервера - всё локально на устройстве

## 📞 Инструкции для пользователей

### Как добавить подписку?

1. Откройте приложение BlowVPN
2. Нажмите **➕ Add Key**
3. Вставьте ссылку подписки:
   ```
   https://example.com/api/v1/client/subscribe?token=xxxxx
   ```
4. Нажмите **Save Subscription**
5. Серверы автоматически загрузятся!

### Как подключиться?

1. На главном экране нажмите большую кнопку ⏻
2. Кнопка станет синей с пульсирующим ореолом
3. Счётчик начнёт отсчёт
4. Вы подключены! 🎉

### Как проверить серверы?

1. Нажмите **🔄 Servers**
2. Потяните вниз чтобы обновить пинги
3. Зелёный пинг = сервер работает
4. Красный N/A = сервер недоступен

## 🐛 Troubleshooting

### Приложение не сохраняет подписку?
- Проверьте, включен ли JavaScript в браузере
- Очистите кеш браузера
- Переустановите приложение

### Пинги показывают N/A?
- Проверьте интернет соединение
- Сервер может быть недоступен
- Попробуйте добавить другую подписку

### Не видно иконки на iOS?
- Убедитесь что используете Safari
- Попробуйте ещё раз: Share → Add to Home Screen
- На iOS нужна iOS 15.1+
